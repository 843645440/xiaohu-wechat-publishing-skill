# Creator Persona Assets

This folder stores the reusable personal IP figure for WeChat covers.

## Character

Name: Future Street Philosopher

Use this figure as the shared visual persona for both accounts when a cover needs a recognizable author mark. It should feel like an unconventional public intellectual: a social observer with a frontier-level technical brain, artistic but credible.

## Files

- `source/future-street-philosopher-concept-sheet.png` - original selected concept sheet.
- `poses/transparent/*.png` - transparent pose cutouts for cover composition.
- `poses/source-keyed/*.png` - green-screen source files used to make transparent cutouts.
- `poses/pose-overview.png` - quick visual overview of all poses.
- `covers/*.png` - example cover compositions.
- `docs/character-spec.md` - stable character identity and usage rules.
- `prompts/reuse-prompts.md` - prompts for generating more poses in the same style.

## Cover Usage

Prefer the transparent pose files for HTML/CSS cover rendering. Keep the character on the right side or lower-right area, and let the article title remain the main visual focus.

Do not generate topic-specific objects into the character image. The character is a reusable author persona, not a literal illustration of the article subject.
